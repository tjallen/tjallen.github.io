360 videos on computer

When watching a 360 video on a computer, the viewing angle is changed by dragging with the mouse.

Make sure you're using the latest version of your web browser (ex. Chrome, Firefox). This 360 video isn't viewable on Safari or Internet Explorer.

360 videos on mobile

On iOS and Android mobile devices, the viewing angle of a 360 video is changed by dragging a finger across the screen or by navigating with the device.

Make sure you're using the latest version of the YouTube app. If you're using an iOS device, it should be an iPhone 4S or newer or an iPad 2 or newer. The iOS device also should be running iOS 7 or later. If you have an Android device, make sure you're running Android 4.3 or later.
